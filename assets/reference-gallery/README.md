# Reference Gallery

This folder is reserved for local, user-provided visual references for ML-SHAP, XGBoost, SHAP, and PDP figure styling.

The public repository does not bundle third-party reference images. Add only images that you created yourself, generated from your own analysis, licensed for redistribution, or have explicit permission to use. Local images in this folder are ignored by Git by default to reduce accidental copyright exposure.

Use reference images to understand preferred visual richness, layout density, color treatment, and figure families. They are style references, not mandatory templates. Keep the workflow's scientific rules in `SKILL.md`: avoid leakage, fit preprocessing only on training data, use SVG/PNG output pairs for new final figures, and do not use SHAP force plots in new analyses unless the user explicitly overrides that rule.
